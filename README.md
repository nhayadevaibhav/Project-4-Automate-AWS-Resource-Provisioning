# Automate AWS Resource Provisioning

## 🎯 Purpose

Automate AWS infrastructure creation using Python scripts instead of manual AWS Console setup.

---

## 🧰 Technologies Used

- Python
- Boto3
- AWS IAM
- AWS EC2
- AWS S3

---

## 🚀 Features

- Create IAM User
- Create S3 Bucket
- Launch EC2 Instance
- Automatic Rollback on Failure
- Fully Automated Infrastructure Setup
